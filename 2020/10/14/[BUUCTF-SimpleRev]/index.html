本来想着自己独立写的，但耗时许久没有成功，取余逆运算是知识盲区。

如：A=(B-C)%D
那么B=(A+C)%D

推倒过程：

​ 比如假设都是正整数
​ A=(B-C)%D
​ 则 B - C = D*n + A 其中 A < D

​ 移项 B = A+C + Dn
​ 当B<D时，两边对D取摸，
​ B = B%D = ( A+C + Dn )%D = (A+C)%D

好家伙，正好最近在学信安数学原理。但需要猜测D的存在。
